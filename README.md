
# Reanimated CSS Examples

## Run the project

### Clone the repo and install dependencies 

```sh
git clone https://github.com/MatiPl01/reanimated-css-examples.git
cd reanimated-css-examples
yarn
npx expo prebuild
```

### Run the app

- iOS
   ```sh
   yarn ios
   ```
- Android
  ```sh
  yarn android
  ```
